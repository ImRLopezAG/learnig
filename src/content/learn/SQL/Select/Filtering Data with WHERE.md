- **Explanation:** The `WHERE` clause is used to filter records.
- **Syntax:**
  ```sql
  SELECT column1, column2, ...
  FROM table_name
  WHERE condition;
  ```
- **Example:**
  ```sql
  SELECT first_name, last_name
  FROM employees
  WHERE department = 'Sales';
  ```
- **Use Cases:** Fetching data that meets specific criteria.
