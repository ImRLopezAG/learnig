- **Explanation:** The `UPDATE` statement is used to modify existing records in a table.
- **Syntax:**
  ```sql
  UPDATE table_name
  SET column1 = value1, column2 = value2, ...
  WHERE condition;
  ```
- **Example:**
  ```sql
  UPDATE employees
  SET department = 'Marketing'
  WHERE last_name = 'Doe';
  ```
- **Use Cases:** Changing existing data.
